## 🚀 시작하기 (3단계)

### 1단계: 워크플로우 활성화 (5분)

**GitHub 웹사이트에서 워크플로우 파일 추가**

👉 **자세한 방법**: `HOW_TO_ENABLE_AUTOMATION.md` 참고

**간단 요약**:
1. GitHub 저장소 → Actions 탭
2. "set up a workflow yourself" 클릭
3. `daily-shorts-workflow.yml` 내용 복사 붙여넣기
4. 커밋!

---

### 2단계: Secrets 설정 (10분)

**API 키와 비밀번호 설정**

👉 **자세한 방법**: `SETUP_ONCE.md` 참고

**필요한 4개 Secrets**:
```
1. PEXELS_API_KEY (Pexels API 키)
2. SMTP_USERNAME (Gmail 이메일)
3. SMTP_PASSWORD (Gmail 앱 비밀번호)
4. RECIPIENT_EMAIL (cogurrl@gmail.com)
```

**설정 위치**: GitHub 저장소 → Settings → Secrets and variables → Actions

---

### 3단계: 첫 테스트 (선택)

**수동으로 테스트 실행**

1. GitHub 저장소 → Actions 탭
2. "🌍 Daily AI Travel Shorts - Full Auto" 클릭
3. "Run workflow" 버튼 클릭
4. 5-10분 대기
5. 이메일 확인!
