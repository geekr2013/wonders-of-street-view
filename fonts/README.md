# 폰트 디렉토리

이 디렉토리에는 쇼츠 자막에 사용할 한글 폰트가 저장됩니다.

## 현재 사용 중인 폰트

- **HakgyoansimYeohaengOTFR.otf**: 학교안심 여행체 (무료 폰트)
  - 출처: [학교안심 여행체](https://gongu.copyright.or.kr/gongu/wrt/wrt/view.do?wrtSn=13407904&menuNo=200020)
  - 라이선스: 자유이용(공공저작물)
  - 사용 용도: 쇼츠 자막 (한글 완벽 지원)

## 폰트 추가 방법

1. 이 폴더에 `.ttf` 또는 `.otf` 파일 추가
2. `scripts/generate_with_pexels.py` 및 `scripts/full_auto_youtube.py`에서 폰트 경로 수정
3. GitHub Actions 워크플로우에서 자동으로 폰트 사용

## 라이선스 확인

- 모든 폰트는 상업적 사용이 가능한 라이선스여야 합니다
- YouTube 업로드 시 저작권 문제가 없는 폰트만 사용하세요
